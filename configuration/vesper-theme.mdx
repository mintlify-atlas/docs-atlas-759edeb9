Vesper is a warm, dark color scheme optimized for long coding sessions. It uses muted oranges, golds, and greens with high contrast text on a deep charcoal background.

## Terminal Configuration

### WezTerm

Complete WezTerm configuration with Vesper theme:

```lua
local wezterm = require 'wezterm'
local config = {}

config.color_scheme = 'Vesper'

config.color_schemes = {
  ['Vesper'] = {
    foreground = '#FFFFFF',
    background = '#101010',
    cursor_bg = '#FFC799',
    cursor_fg = '#101010',
    cursor_border = '#FFC799',
    selection_bg = 'rgba(50% 50% 50% 50%)',
    scrollbar_thumb = 'rgba(50% 50% 50% 50%)',
    split = '#505050',
    
    ansi = {
      '#101010', '#F5A191', '#90B99F', '#E6B99D',
      '#ACA1CF', '#E29ECA', '#EA83A5', '#A0A0A0',
    },
    brights = {
      '#7E7E7E', '#FF8080', '#99FFE4', '#FFC799',
      '#B9AEDA', '#ECAAD6', '#F591B2', '#FFFFFF',
    },
  }
}

return config
```

See [WezTerm Configuration](/configuration/wezterm) for complete setup.

```powershell
--color=marker:#FFC799,fg+:#ffffff,prompt:#FFC799,hl+:#FFCFA8
--height 50%
--layout reverse
--border rounded
"@
```

## Design Philosophy

### Readability First

Vesper prioritizes long-term readability:

- **High contrast** (`#FFFFFF` on `#101010`) for main text
- **Muted accents** to reduce eye strain
- **Warm palette** for comfortable extended use
- **Consistent semantics** (errors always red, success always green)

### Color Psychology

| Color | Meaning | Usage |
|-------|---------|-------|
| Orange (`#FFC799`) | Action, emphasis | Keywords, cursors, primary UI |
| Green (`#90B99F`) | Success, addition | Git additions, successful operations |
| Red (`#F5A191`) | Error, deletion | Errors, git deletions, warnings |
| Cyan (`#99FFE4`) | Data, strings | String literals, data values |
| Purple (`#ACA1CF`) | Functions | Callable entities |
| Gold (`#FFCFA8`) | Types | Type definitions, classes |

## Accessibility

### Contrast Ratios

| Foreground | Background | Ratio | WCAG Level |
|------------|------------|-------|------------|
| `#FFFFFF` | `#101010` | 19.5:1 | AAA |
| `#FFC799` | `#101010` | 10.2:1 | AAA |
| `#99FFE4` | `#101010` | 14.8:1 | AAA |
| `#505050` | `#101010` | 3.2:1 | AA |

<Note>
  All primary text colors meet WCAG AAA standards for contrast.
</Note>
